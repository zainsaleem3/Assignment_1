Final Project (Introduction to Programming for IT)

Created a project that develops a student ledger and displays details about the students. The program allows the admin to save the entered details: Yes/No. If ‘Yes', it allows the admin to add more students: Yes/No. If ‘No', it displays all the entered student details in form of a table where first column is “Student Number”, second column is “First Name” and so on. From here, admin is allowed to go to the main menu and exit the program.
