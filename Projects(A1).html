Projects
Final Project (Introduction to Programming for IT)

Created a project that develops a student ledger and displays details about the students. The program allows the admin to save the entered details: Yes/No. If ‘Yes', it allows the admin to add more students: Yes/No. If 'No', it displays all the entered student details in form of a table where first column is “Student Number”, second column is “First Name” and so on. From here, admin is allowed to go to the main menu and exit the program.

Final Project (Object-Oriented Programming)

I needed to develop a game that converted normal English words into secret codes. In order to convert, the program randomly applied an encryption algorithm to any given message. The algorithms I needed to implement included Substitution, Playfair, Caesar, Transposition, Product, and RSA ciphers.

Final Network Design (Introduction to Networks)

With a given network topology, we were to create subnets and addresses for various devices (routers, switches, endpoint devices) within Local Area Networks that conjoined into one Wide Area Network and made sure that they were all able to ping each other. If not, then were to troubleshoot each device and its configuration to verify and update its configuration in order for certain devices to communicate with one another.
